Comments on the course
I went into this class with some Python experience, which I really think helped me do as well as I did in this class. Lots of little projects for weekly assignments, I took this class once a week in a 3 hour block, which really made lectures feel like they went on forever.

Basic Python information, I highly recommend taking notes and reviewing vocabulary if it's your first time looking at/writing code.
Comments on the professor
Saweh's lectures put me to sleep. Again, I did take this class in a 3 hour block, from 6-9pm which I definitely would not recommend. The class itself wasn't terrible, but I did not get a lot out of Saweh's lectures.
